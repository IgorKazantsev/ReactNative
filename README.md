# React Native Quiz App

React Native viktoriinirakendus, mis kasutab Open Trivia DB API-t küsimuste laadimiseks.

## Funktsionaalsus

- Küsimuste laadimine Open Trivia DB API kaudu
- Kategooria valik
- Raskusastme valik
- Küsimuse tüübi valik:
  - Multiple Choice
  - True / False
- Küsimuste kuvamine ükshaaval
- Taimer iga küsimuse jaoks
- Vastuste kontroll
- Tulemuste arvutamine
- Edukuse protsent
- Testi kestuse arvutamine
- Kasutaja nime sisestamine
- Leaderboard TOP 5 tulemustega
- SQLite andmebaas tulemuste salvestamiseks
- localStorage tugi veebiversiooni jaoks

## Kasutatud tehnoloogiad

- React Native
- Expo
- Expo Router
- TypeScript
- SQLite
- Open Trivia DB API

## Käivitamine

```bash
npm install
npx expo start
```
